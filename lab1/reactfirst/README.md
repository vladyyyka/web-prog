Лабораторная работа 1 по web-программированию, выполнил студент МТ-302.


В папке server находится файл app.js, который создает серверное приложение на Node.js с использованием Express.js. Оно обрабатывает запросы для выполнения базовых арифметических операций (сложение, вычитание, умножение, деление) и возвращает результаты в формате JSON.


В папке src находятся файлы App.js, который создает калькулятор на React, который позволяет пользователю выполнять арифметические операции (сложение, вычитание, умножение, деление). Он отправляет запросы на сервер для вычислений, отображает результаты или ошибки, а также файл App.css, который используется для стилей файла App.js.
